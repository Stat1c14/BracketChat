    Welcome to BracketChat!


    Hey, this is Chandler, the creator of BracketChat. BracketChat is a program designed to allow code to be shared accross large groups of people instantaneously. It is meant to be used in classroom/conference settings to quickly exchange code snippets amoungst participants without having to share personal information. It works by using Node.js and the socket.io library to asynchronously transport data between the end users and the server. This process occurs seemlessly and allows for a high number of simultaneous connections as well as near-instant message delivery times.
